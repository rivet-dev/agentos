TypeScript is built into JavaScript execution. There is no standalone
TypeScript package or `/typescript` subpath. Create a `JavaScriptRuntime` and
use its `typescript` property, or call the flat TypeScript methods on `AgentOs`.

## Install

```bash
pnpm add @rivet-dev/agentos-javascript
```

## API

The focused package exposes `typescript.execute()`, `evaluate()`,
`executeFile()`, `check()`, and `checkProject()`. The corresponding core
methods are `executeTypeScript()`, `evaluateTypeScript()`,
`executeTypeScriptFile()`, `checkTypeScript()`, and
`checkTypeScriptProject()`.

Execution transpiles without semantic type checking. Checking is always
explicit and returns structured diagnostics. Inline JavaScript and TypeScript
share retained state when they use the same execution ID.

See the [JavaScript execution guide](/docs/execution/javascript).
